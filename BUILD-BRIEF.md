- Food dishes (wings, pasta, burgers, etc.)

- Exterior shots (storefronts)
- Detail shots (menus, beverages, decor)

## Key Requirements
1. LIGHT MODE ONLY - no dark mode
2. NO EMOJIS anywhere
3. Mobile-first responsive design
4. Menufy order buttons prominent
5. OpenTable reservation buttons prominent
6. Remove all references to closed restaurants (Red/Osteria cross-promo)
7. SEO optimized with structured data (LocalBusiness schema)
8. SSL ready

## Site Structure (Each Restaurant)
- Home (hero photo, tagline, CTA buttons)
- Menu (full menu from Menufy, organized by section)
- About (story, interior photos)
- Gallery (photo grid)
- Contact (address, phone, hours, map embed)
- Order Online (link to Menufy)
- Reservations (link to OpenTable)

## Design Notes
- Andrew dislikes emojis - never use them
- Light backgrounds only
- The brand is monochrome/woodcut for Sapsuckers - lean into that aesthetic
- Cafe Red should feel warmer, more upscale Italian
